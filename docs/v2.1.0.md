## 🆕 Added

### modules/database (신규 모듈)

| 리소스 | 이름 | 상세 |
|--------|------|------|
| RDS Instance | `everybuddy-mysql` | MySQL 8.0.44, db.t3.micro |
| DB Subnet Group | - | private-db-a (AZ-a), private-db-b (AZ-b) |
| Security Group | `everybuddy-rds-sg` | 3306 — 백엔드 SG만 허용 |

### 접속 정보
- **Endpoint:** `everybuddy-mysql.cno2sciake0k.ap-southeast-1.rds.amazonaws.com:3306`

- **AZ:** ap-southeast-1a

---

## 보안 설계

- RDS는 Private Subnet에만 배치 (외부 직접 접근 불가)
- DB Subnet Group에 Multi-AZ 서브넷 포함 (AZ-a, AZ-b) — 추후 Multi-AZ 전환 대비
- 3306 포트는 백엔드 SG에서만 접근 허용

---

## 아키텍처

```
EC2 everybuddy-backend (:3306)

  ▼
Private DB Subnet 10.0.21.0/24 (AZ-a)
  └── RDS everybuddy-mysql (MySQL 8.0)
```
