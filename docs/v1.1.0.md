## 🆕 Added

### Networking
- **Subnet** `everybuddy-public-subnet-monitoring` — 10.0.2.0/24, ap-southeast-1a

### Compute
- **EC2** `everybuddy-monitoring` — t3.micro, Ubuntu 24.04 LTS, ap-southeast-1a

### Security
- **Security Group** `everybuddy-monitoring-sg`
  - Inbound: 22 (SSH), 3000 (Grafana), 9090 (Prometheus) — 0.0.0.0/0
  - Inbound: 3100 (Loki) — 백엔드 SG (`everybuddy-backend-sg`)만 허용
  - Outbound: 전체 허용
- **Backend SG 추가 규칙**
  - Inbound: 9100 (Node Exporter) — 모니터링 SG만 허용
  - Inbound: 8080 (Actuator) — 모니터링 SG만 허용 (명시적 추가)
