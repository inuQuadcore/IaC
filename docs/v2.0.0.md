## 🆕 Added

### Terraform 모듈 구조 도입

단일 tf 파일 구조에서 모듈 분리로 전환.

```
modules/
├── networking/    # VPC, Subnet, IGW, NAT GW, Route Table
├── security/      # Security Groups
├── compute/       # EC2, Key Pair, EIP
└── storage/       # S3
```
