# v2.6.0 — Terraform Remote Backend (S3) 설정

**날짜:** 2026-03-13
**커밋:** `86c091d` Feat: S3 Remote Backend 설정

---

## 🆕 Added

- **S3 Bucket** `everybuddy-terraform-state` — Terraform State 저장소

## ✏️ Changed

- `versions.tf` — backend 블록 추가

```hcl
terraform {
  backend "s3" {
    bucket = "everybuddy-terraform-state"
    key    = "terraform.tfstate"
    region = "ap-southeast-1"
  }
}
```

---

## 변경 이유

| 항목 | 로컬 State | S3 Remote State |
|------|-----------|-----------------|
| 팀 공유 | 불가 | 가능 |
| 유실 위험 | 높음 | 낮음 (S3 내구성 99.999999999%) |
| 동시 작업 | 충돌 위험 | DynamoDB Lock으로 방지 가능 |
