## 🆕 Added

### Networking
- **VPC** `everybuddy-vpc` — 10.0.0.0/16, DNS hostname/support 활성화
- **Internet Gateway** `everybuddy-igw` — VPC 인터넷 연결
- **Subnet** `everybuddy-public-subnet` — 10.0.1.0/24, ap-southeast-1a, Public IP 자동 할당
- **Route Table** `everybuddy-public-rt` — 0.0.0.0/0 → IGW

### Compute
- **EC2** `everybuddy-backend` — t3.micro, Ubuntu 24.04 LTS, ap-southeast-1a
- **Key Pair** `server-key` — serverkey.pub 등록

### Security
- **Security Group** `everybuddy-backend-sg`
  - Inbound: 22 (SSH), 80 (HTTP), 443 (HTTPS), 8080 (Spring Boot) — 0.0.0.0/0
  - Outbound: 전체 허용

### Storage
- **S3 Bucket** `everybuddy-files-prod-20250103`
  - 파일 업로드/다운로드용
  - Versioning 활성화
  - Public Access 전체 차단
  - CORS: GET/PUT/POST/DELETE/HEAD 허용
