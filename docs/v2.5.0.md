# v2.5.0 — 모니터링 Security Group 보완

**날짜:** 2026-03-13
**커밋:** `dc9cfd7` Feat: monitoring 보안그룹 설정

---

## ✏️ Changed

### modules/security — 모니터링 SG 규칙 추가

Private Backend 전환 이후 Monitoring 서버가 Private Backend에 접근할 수 있도록 SG 규칙 보완.

| 방향 | 포트 | 허용 대상 | 용도 |
|------|------|-----------|------|
| Monitoring → Private Backend | 9100 | `everybuddy-monitoring-sg` | Node Exporter 스크랩 |
| Private Backend → Monitoring | 3100 | `everybuddy-private-backend-sg` | Promtail → Loki 로그 전송 |

---

## 변경 이유

v2.4.0에서 백엔드를 Private Subnet으로 이전하면서 기존 SG 규칙이 Public Backend(`everybuddy-backend-sg`)를 참조하고 있었음.
Private Backend SG(`everybuddy-private-backend-sg`)를 참조하도록 규칙 갱신.
