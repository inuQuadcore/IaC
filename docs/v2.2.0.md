## 접근 구조

```
[GitHub Actions / 개발자]
        │
        │ SSH (22)
        ▼
Bastion EC2 (public-b, AZ-b)
        │
        │ SSH ProxyJump
        ▼
Private Backend EC2 (10.0.11.x, private-app-a, AZ-a)
```

## CI/CD 배포 흐름

```
GitHub Actions
  → SCP(Bastion ProxyJump) → docker-compose.yml 전송
  → SSH(Bastion ProxyJump) → docker compose up -d
```
